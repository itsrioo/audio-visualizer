Audio Visualizer - Rio Donald

A space themed audio visualizer using waveform visualization

GITHUB LINK:
https://itsrioo.github.io/audio-visualizer/


Features
- Play/Pause Controls
- Volume Slider (0%-100%)
- Audio waveform using p5.FFT
- Space themed design with stars, planets, and animations

How to use
1. Click the play button to start
2. Drag the moon slider to adjust the volume
3. Watch the visualizer
